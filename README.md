# LOGICSOFT — Sitio web oficial

Este sitio es la página oficial de **LOGICSOFT**, simulador educativo de lógica tipo PLC (PWA), creado por **PASIR INDUSTRIAL**.

- `index.html` — Página principal / presentación
- `privacy.html` — Política de Privacidad (enlace público para Google Play u otros)
- `styles.css` — Estilos del sitio

## Publicación en GitHub Pages
1. Crea un repositorio, por ejemplo: `LOGICSOFT_SITE`.
2. Sube estos archivos a la rama `main`.
3. Ve a **Settings → Pages** y selecciona:
   - Source: **Deploy from a branch**
   - Branch: **main** (folder **/** root)
4. Tu enlace público será:
```
https://TU_USUARIO.github.io/LOGICSOFT_SITE/
```
y la política en:
```
https://TU_USUARIO.github.io/LOGICSOFT_SITE/privacy.html
```
